# 测试用例集合

## 测试用例1：基础多级标题

输入：
```markdown
# 第一章
## 第一节
### 第一小节
### 第二小节
## 第二节
# 第二章
```

期望输出：
```markdown
# 1. 第一章
## 1.1. 第一节
### 1.1.1. 第一小节
### 1.1.2. 第二小节
## 1.2. 第二节
# 2. 第二章
```

## 测试用例2：包含代码块的内容

输入：
```markdown
# 代码示例

这是一个代码块：

```python
# 这不是标题
def hello():
    print("Hello World")
```

## 函数说明
```

期望输出：
```markdown
# 1. 代码示例

这是一个代码块：

```python
# 这不是标题
def hello():
    print("Hello World")
```

## 1.1. 函数说明
```

## 测试用例4：跳级标题

输入：
```markdown
# 第一章
### 直接到三级标题
## 回到二级标题
#### 四级标题
```

期望输出：
```markdown
# 1. 第一章
### 1.0.1. 直接到三级标题
## 1.1. 回到二级标题
#### 1.1.0.1. 四级标题
```
